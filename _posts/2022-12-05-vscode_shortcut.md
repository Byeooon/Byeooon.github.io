###### Visual Studio Code에서 사용할 수 있는 단축키들에 대해 간단하게 정리해보도록 하겠습니다.

###### (Mac OS 기준으로 작성)

```
command⌘ + D
같은 단어를 연속으로 선택
```

```
command⌘ + Shift + 방향키
한 행을 모두 선택
```

```
Shift + 방향키
한 칸씩 방향키 방향으로 이동하면서 선택
```

```
command⌘ + F
원하는 단어 찾기
```

```
command⌘ + /
주석 설정
```
