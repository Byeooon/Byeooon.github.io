###### Github의 checkout과 stash명령어 사용법에 대해 알아보도록 하겠습니다.

* 해당 브랜치로 전환

```python
!git checkout [브랜치명]
```

* 원격저장소에 있는 내용을 로컬에 pull하기전에 충돌을 막기위해 로컬의 내용을 임시 저장하는 명령어.

```python
!git stash
```
