- **디렉토리 목록확인**
```
$ ls
```
- **디렉토리 변경**
```
$ cd
$ cd .. : 상위 디렉토리로 이동
```
- **텍스트 파일의 내용보기**
```
$ cat
```
- **파일의 존재 여부 확인**
```
$ which
```
- **디렉토리 생성**
```
$ mkdir [디렉토리명]
```
- **디렉토리 삭제**
```
$ rm -r [디렉토리명]
```
- **파일삭제**
```
$ rm [파일명]
```
- **현재 디렉토리 확인**
```
$ pwd
```
- **터미널 창 초기화**
```
$ clear
```
- **비밀번호 설정**
```
$ sudo passwd [사용자명]
```
