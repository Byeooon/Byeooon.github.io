###### 오늘은 리눅스의 에디터 중 하나인 vi에서 자주 쓰이는 명령어들을 간단하게 정리해보도록 하겠습니다.


###### 입력 모드

```
'i' or 'a' 를 눌러서 입력 모드로 진입 가능
```

* 이동 : 방향키 이용 ( 방향키 없는 경우 : h, j, k, l )
* 입력 : 키보드 이용
* x : 커서 위치 글자 삭제
* dd : 커서 라인 전체 삭제


###### 명령 모드

```
입력 모드에서 'ESC'를 눌러서 전환 가능

':' 를 눌러서 명령어를 입력할 수 있다.
```

* :q -> 종료
* :q! -> 강제 종료
* :wq -> 저장 후 종료
