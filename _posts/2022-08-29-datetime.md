###### python의 threading을 이용하여 정해진 시간마다 반복적으로 시간을 출력하는 코드입니다.
```python
import threading
import datetime

def repeat_output():
    now = datetime.datetime.now()
    print(now)
    threading.Timer(3, repeat_output).start()
    
repeat_output()
```
